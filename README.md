# Smart Notice Board using Pixel LED and Bluetooth

## 📌 Project Overview
The Smart Notice Board is an electronic display system that uses Pixel LED technology and Bluetooth communication to display messages wirelessly.

## 🔧 Components Used
- Pixel LED Display
- Bluetooth Module
- Microcontroller
- Power Supply

## ⚙️ Working
Messages are sent via Bluetooth from a mobile device to the microcontroller.
The microcontroller processes the message and displays it on the Pixel LED board.
